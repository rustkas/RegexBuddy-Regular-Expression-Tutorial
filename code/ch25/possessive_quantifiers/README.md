possessive_quantifiers
=====

## Create new project

----	
	
	# all commands in one string
	$ rebar3 new lib possessive_quantifiers && cd possessive_quantifiers
	
## Get dependencies
	$ rebar3 deps	

## Format
	$ rebar3 format
	
## EUnit
-----
	$ rebar3 eunit
	$ rebar3 eunit -m possessive_quantifiers_tests

An OTP library

Build
-----

    $ rebar3 compile
