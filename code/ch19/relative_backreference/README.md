relative_backreference
=====

## Create new project

----	
	
	# all commands in one string
	$ rebar3 new lib relative_backreference && cd relative_backreference
	
## Get dependencies
	$ rebar3 deps	

## Format
	$ rebar3 format
	
## EUnit
-----
	$ rebar3 eunit
	$ rebar3 eunit -m relative_backreference_tests

An OTP library

Build
-----

    $ rebar3 compile
