repetions
=====

## Create new project

----	
	
	# all commands in one string
	$ rebar3 new lib repetions && cd repetions
	
## Get dependencies
	$ rebar3 deps	

## Format
	$ rebar3 format
	
## EUnit
-----
	$ rebar3 eunit
	$ rebar3 eunit -m repetitions_tests

An OTP library

Build
-----

    $ rebar3 compile
